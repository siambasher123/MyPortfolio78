My Portfolio Website :
This is my personal portfolio website built with PHP, HTML, CSS, and JavaScript. It showcases my skills, projects, and experience as a web developer in a clean, responsive, and interactive design.

Features -> 
Home Page: Introduction with animated hero section.
About Me: Highlights my skills, experience, and education.
Projects: Displays all my projects with images, descriptions, and GitHub links.
Project Details: Click on a project to see its description and access the GitHub repository.
Responsive Design: Works on desktop, tablet, and mobile devices.
Visit Counter: Tracks visitors  visits using cookies and session.

Technologies Used ->
PHP
MySQL (Database for projects and about info)
HTML5 & CSS3
JavaScript 

How to Use ->
Clone this repository to your local server.
Import the database using phpMyAdmin or any MySQL client.
Update the db.php file with your database credentials.
Open the website in your browser.
